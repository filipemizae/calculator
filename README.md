# 🧮 Calculator

A simple calculator with a **Java Spring Boot** backend and a **vanilla JavaScript** frontend.

Uma calculadora simples com backend em **Java Spring Boot** e frontend em **JavaScript puro**.

---

## 🌐 Live Demo
👉 [https://calculator-nu-hazel-75.vercel.app/](https://calculator-nu-hazel-75.vercel.app/)

---

## 🛠️ Tech Stack / Tecnologias

| Frontend | Backend | Deploy |
|---|---|---|
| HTML, CSS, JavaScript | Java 17, Spring Boot | Vercel + Railway |

---

## ⚙️ Features / Funcionalidades

- ➕ Addition / Adição
- ➖ Subtraction / Subtração
- ✖️ Multiplication / Multiplicação
- ➗ Division / Divisão
- 🗑️ Clear all / Limpar tudo
- ⌫ Delete last digit / Apagar último dígito

---

## 🚀 How to run locally / Como rodar localmente

### Backend
```bash
cd calculator
mvn spring-boot:run
```

### Frontend
Open `index.html` directly in the browser or move it to `src/main/resources/static/`.

Abra o `index.html` diretamente no navegador ou mova para `src/main/resources/static/`.
